Nodejs


Node is a server environment that uses Javascript. It has become a popular tool since it can execute Javascript outside a browser and thus be used both frontend and backend.
Node has a built tool called npm which stands for Node Package Manager. With this we can easily install modules to use for easy development. Framework modules can be found on npmjs.com.
To install a module first create a json file named package.json, then run the following command in your project’s folder:

npm install [module name]


This will generate a node_modules folder with all modules in it and it will update the package.json file where it’s possible to see all dependencies. You can also manually add the dependencies to the package file.
Then you will only need to run npm install in the command line.
To run a node program simply just type the following in the command line in your project’s folder:

node [name of js file]


Consider using Nodemon for an easier execution.
