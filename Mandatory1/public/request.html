Request


Request is a module for Node handling http requests. So we use it when we need to interact with other web services like an API call.
After installing the module, we import it and make a request to a web page.

Example


// importing it
const request = require(“request);
// using request to display the google.com’s html page when accessing the route proxygoogle
app.get(“/proxygoogle”, (req, res) => {
request.(http://www.google.com, (error, response, body) => {
return res.send(body);
});
});
