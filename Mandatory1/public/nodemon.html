Nodemon


Nodemon is a tool for Node and replaces the command node with nodemon when starting the app. With nodemon you only need to start your app once and then it will restart the app every time there has been a change in the code.
This saves time and is handy for developing web apps where you want to see how your change has affected your page.
You would almost always need Nodemon for all your projects and therefore you should install it globally with the following command:

npm install nodemon -g
