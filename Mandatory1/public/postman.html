Postman


Postman is an API tool you can use when developing APIs or when testing an API that you will use in your own app. With that you can connect to an API and use the different http requests.
For more about APIs, see API.

Verb Example Description
GET /collection/id This is how you receive information.
You either ask for the whole collection or a specific item with an id
POST /collection This is how you add an item to a collection.
Trust that the developer takes care of the id
PUT /collection/id This is how you update an item in a collection.
DELETE /collection/id This is how you delete an item in a collection.


When working with someone's API, you will most likely not be able to do anything else than GET from Postman.
But when developing your own API, Postman is a great tool to test that you can actually POST, PUT and DELETE.
