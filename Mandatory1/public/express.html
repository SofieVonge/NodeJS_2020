Express


Express is a module for Node.js. It’s a web framework and therefore used in web applications. It’s popular because it’s minimalistic and very flexible.
After installing it, it has to be imported to the controllerside of the app in order to be able to use it. After that you make your app listen on a specific port and then you can create various http verb requests like get and post.

This is how to set it up in your code:

// importing it
const express = request(“express”);

// express returns a method
const app = express();

// tell express the name of the static folder
app.use(express.static('public'));

// app listens on port 3000. It’s also possible to give listen a callback function in order to check for errors.
app.listen(3000);

Here are some examples on various get methods.


//a get method for the index page sending a JSON object
app.get(“/”, (req, res) => {
return res.send({message: “sending a JSON object”});
});

//another get method, this time for the route aboutus sending a html file
//located in the static folder public
app.get(“/aboutus”, (req, res) => {
return res.sendFile(__dirname + /public/aboutus.html);
});
//a get method with a path variable
app.get(“/collection/:id”, (req, res) => {
const item = array.find(item => item.id === Number(req.params.id));
return res.send({response: item});
});
