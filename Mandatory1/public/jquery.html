jQuery


jQuery is a Javascript library which makes Javascript much easier to use. It uses the DOM like Javascript itself and also has AJAX. jQuery also handles all cross-platform issues.
To use the library, you need to run a script. This script can be run locally, but it’s better to use someone else’s hosting of it since you’re then saved the trouble of sending the script with every request.
You can get the script from Google here.
When using Google’s hosting, you’re sure to have faster loading of your web page, since Google’s servers are everywhere.
With jQuery you select html elements and do actions on them very easily. You select them with the $ sign and one or more selectors which can be for instance a class, id or type.

Example of how it can look like:

${#button}.click(() => {
// code for what happens when you click on the element with the id of button
});

You'll most likely use jQuery when you handle different events on the page, for instance clicking a button or hover the mouse over something.
